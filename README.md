# Graph Theory & Optimization Algorithms 🕸️

This repository contains a collection of projects, laboratory assignments, and exercises focused on **graph theory**, **structural analysis**, and **optimization problems**. The code was developed in Python using Jupyter Notebooks (Google Colab) during my Master's degree studies in Applied Computer Science.

## 📂 Repository Structure

The repository is organized into directories, each covering a specific topic or business problem solved using graph algorithms:

*   **`/Analiza statystyczna grafów`** - Statistical analysis of networks and graphs.
*   **`/Grafowe problemy optymalizacyjne`** - Solutions to various graph-based optimization problems.
*   **`/Harmonogram produkcji`** - Production scheduling using graph theory concepts.
*   **`/lab3`** & **`/lab4`** - Academic laboratory exercises focusing on specific graph algorithms.
*   **`/Maksymalizacja zysku`** - Algorithms focused on profit maximization models.
*   **`/Planowanie kierunku zadanie`** - Direction and path planning tasks.
*   **`/Strukturalne właściwości grafów`** - Exploring the structural and topological properties of different graph models.
*   **`/projekt`** - The main academic project for the course.
